## ğŸ“¦ Installation

### Prerequisites
- Node.js (v20.19.0 or >=22.12.0)
- pnpm (9.14.2 or higher)

### Setup Steps

1. **Clone the repository**
```sh
git clone <your-repo-url>
cd notebook
```

2. **Install dependencies**
```sh
pnpm install
```

3. **Configure Supabase**

Create a `src/supabase.js` file with your credentials:
```javascript
import { createClient } from '@supabase/supabase-js'

const supabaseUrl = 'YOUR_SUPABASE_URL'
const supabaseAnonKey = 'YOUR_SUPABASE_ANON_KEY'

export const supabase = createClient(supabaseUrl, supabaseAnonKey)
```

4. **Create Database Table**

Run this SQL in Supabase SQL Editor:
```sql
create table word_table (
  id bigint generated by default as identity primary key,
  english_word text not null,
  turkish_word text not null,
  english_description text,
  turkish_description text,
  select boolean default false,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);

-- Enable Row Level Security
alter table word_table enable row level security;

-- Create policies
create policy "Enable read for all users" on word_table
  for select using (true);

create policy "Enable insert for all users" on word_table
  for insert with check (true);

create policy "Enable update for all users" on word_table
  for update using (true) with check (true);

create policy "Enable delete for all users" on word_table
  for delete using (true);
```

5. **Start development server**
```sh
pnpm dev
```

### Navigation
- **Cards** - Main page with flip cards
- **Words** - Form to add new words + Table view

### Adding Words
1. Navigate to **Words** page
2. Fill in English and Turkish words in the form
3. Click **Submit**
4. Words appear immediately in the table below

### Using Favorites
1. Click the checkbox on any card to mark as favorite
2. Click the **Favorites** button to filter
3. Click again to show all words
4. Favorites appear in the **Test** carousel

### Search & Filter
- Use the search bar to filter words instantly
- Works across both English and Turkish fields
- Filters apply to current view (all or favorites only)

## ğŸ› Troubleshooting

### Supabase Connection Issues
- Verify your URL and anon key in `supabase.js`
- Check that RLS policies are enabled
- Ensure table name matches (`word_table`)

### Cards Not Showing
- Check browser console for errors
- Verify Supabase table has data
- Try refreshing the page

### Flip Animation Not Working
- Ensure Quasar CSS is properly imported in `main.js`
- Check that card components have proper height set
